Note:
	Make sure to first compile all the provided java files
	- Project3.java
	- scheduler.java
	- jobs.java
	- compareJobs.java
	- RR.java
	- SRT.java
	- FB.java
	Type the following to compile all the java files:
	- "javac *.java"
	
	When running the Scheduling Algorithm Simulation type the following:
	-"java Project3 RR"  for round-robin
	-"java Project3 SRT" for shortest remaining time
	-"java Project3 FB"  for feedback
	-"java Project3 ALL" for all three algorithms
